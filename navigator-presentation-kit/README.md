# Navigator Presentation Kit

Переносной набор для создания HTML-слайдов в стиле «Крафтовый терминал». Презентации заточены под трансляцию зрителям во время записи YouTube-видео или эфиров.

## Зачем

Когда записываешь видео и хочешь, чтобы зритель видел структурированные слайды на экране — а не PDF, не Keynote, не Google Slides. Получаешь:

- Полноэкранные слайды со scroll-snap (один слайд = один экран)
- Прогресс-бар, счётчик «05 / 26», навигация стрелками
- Полный экран по `F`
- Готовые компоненты: цитаты экспертов, кейсы «было/стало», сравнительные таблицы, терминалы с командами, лид-магнит в финале
- Никакого React, никакой сборки — один HTML-файл, открывается двойным кликом

## Быстрый старт

### Вариант 1 — через AI-агента

```bash
# Скопируй кит в новый проект
git clone https://github.com/USERNAME/navigator-presentation-kit.git my-presentation
cd my-presentation

# Открой Claude Code / Cursor / Copilot в этой папке
# Скажи агенту:
```

> Собери презентацию по теме «Как настроить деплой на Vercel за 10 минут» для аудитории начинающих фронтенд-разработчиков. Видео 12 минут.

Агент прочитает `CLAUDE.md`, поймёт правила, и соберёт `navigator.html` под твою тему.

### Вариант 2 — вручную

1. Открой `navigator-template.html` в редакторе
2. Скопируй блок `<style>` целиком (это твоя дизайн-система)
3. Замени контент `<section>`-ов на свой по типам:
   - Титульный (большая цифра + название)
   - Слайд с буллитами
   - Слайд-цитата эксперта (тёмный фон)
   - Слайд-кейс (две колонки «было / стало»)
   - Слайд с командой (терминал)
   - Сравнительная таблица
   - Финальная цитата (тёмный фон, полный экран)
   - Лид-магнит / CTA
4. Сохрани как `navigator.html`, открой в браузере

Подробности — в [`navigator-guide.md`](navigator-guide.md).

## Управление презентацией

| Клавиша | Действие |
|---|---|
| `→` `↓` `Space` `PageDown` | Следующий слайд |
| `←` `↑` `PageUp` | Предыдущий слайд |
| `Home` / `End` | К первому / последнему слайду |
| `F` | Полный экран |

## Дизайн-система

Стиль называется «Крафтовый терминал» — пергамент, терракот, графит, белый. Тёплый ретро-минимализм, никаких градиентов кроме тёмных слайдов.

**Палитра:**
- `#F5E6D0` пергамент
- `#E8762B` терракот
- `#1E1E1E` графит
- `#FFFFFF` белый
- `#C84B3A` боль / ❌
- `#7A8C3A` решение / ✅
- `#B8841A` золото для инсайтов

**Шрифты (Google Fonts):**
- `Oswald` 700 — заголовки
- `Press Start 2P` — пиксельные цифры и теги
- `Inter` — основной текст
- `JetBrains Mono` — код и команды

## Готовые компоненты

В эталоне реализованы:

- **Hero с огромной цифрой** — для титула (43.000 установок / 65 строк / 100K пользователей)
- **Буллиты** — список 3–5 пунктов с пиксельными цифрами 01-05
- **Принцип-слайд** — большая цифра принципа + название + формулировка в карточке «📝 ЗАПОМНИ»
- **Цитата эксперта** — тёмный слайд с большим открывающим «и атрибуцией
- **Кейс «было / стало»** — двухколоночный слайд с красным/зелёным бордером
- **Терминал** — Mac-style окно с командой
- **Сравнительная таблица** — с тёмной шапкой и цветной подсветкой
- **Акцентный слайд** — терракотовый градиент для вступлений к секциям
- **Финальная цитата** — на полный экран
- **Лид-магнит** — три карточки артефактов + CTA-блок

## Структура папки ролика

В рабочем процессе кит — часть пакета файлов на каждое видео:

```
your-video/
├── scenario.md         ← полный сценарий с тайминогом и open loops
├── teleprompter.md     ← разговорный текст для ведущего
├── teleprompter.html   ← HTML-телесуфлёр с автопрокруткой
└── navigator.html      ← ЭТА презентация для трансляции зрителям
```

Каждый файл — для своей задачи. `navigator.html` транслируется в OBS как источник, остальные открыты у ведущего на втором мониторе.
